1.  .env 파일을 만들고 키를 넣으세요
   OPENAI_API_KEY="키를 입력하세요"
 
2.  아래 링크에서 다운받고 깃파일 폴더에 넣으세요 
   https://drive.google.com/drive/folders/1hl8hHsRAcUd3AVUK-Tibb6yESouYPvXx?usp=sharing
DB Data chroma_db_combined_... 폴더 (로컬 데이터베이스)
    
3.   라이브러리 설치
     "pip install -r requirements.txt"
     (추가로 필요한 라이브러리가 자동으로 설치 될 수 있습니다) 
     
4.  프롬프트에서 "python api.py" 실행
   이 조건들이 갖춰진다면 http://localhost:8000 접속 시 동일한 화면을 볼 수 있습니다.
   (GPU VRAM이 최소3GB가 필요합니다)
